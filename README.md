## 安装
![效果图](http://www.ml520.cn/json_test.gif)
```ruby
pod 'HBChatKeyBoard'
```
## 须知
1. 录音文件是`.caf`格式，如需`.mp3`自行通过三方框架`lame.framework`转换
2. 表情资源目前***仅支持本地！***
3. 如需更改输入框样式，请自行组装即可。


## 功能
1. iPhone X适配
2. 语音，表情以及更多功能 
3. 支持长按删除，输入框内容高度自适应
4. 整体逻辑与微信一致
